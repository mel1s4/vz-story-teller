== Description ==

VZ Story Teller is an open-source WordPress plugin that brings professional screenwriting capabilities to your WordPress site. Write, format, and manage screenplays with industry-standard elements like sluglines, action lines, dialogue, and transitions—all within your familiar WordPress environment.

Perfect for filmmakers, screenwriters, content creators, and anyone who needs to write formatted scripts without leaving their WordPress workflow.

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins → Add New
3. Search for "VZ Story Teller"
4. Click "Install Now"
5. Click "Activate"

= Manual Installation =

1. Download the plugin ZIP file from the WordPress.org plugin directory
2. Log in to your WordPress admin panel
3. Navigate to Plugins → Add New
4. Click "Upload Plugin"
5. Choose the ZIP file and click "Install Now"
6. Click "Activate"

= After Activation =

1. Navigate to Scripts → Add New in your WordPress admin
2. Enter a title for your script
3. Optionally set a featured image (cover image)
4. Click "Edit in Frontend" to open the script editor
5. Start writing your screenplay!

= Creating Your First Script =

1. Go to **Scripts → Add New** in your WordPress admin panel
2. Enter a title for your script
3. Optionally set a featured image (cover image)
4. Add categories or tags for organization
5. Click the **"Edit in Frontend"** button in the publish box
6. Use the element selector to add different script components
7. Click on any element to edit its content
8. Drag to reorder elements
9. Changes are automatically saved as you type

== Frequently Asked Questions ==

= Do I need to know how to code? =

No! VZ Story Teller is designed for non-technical users. The editor is intuitive and easy to use. If you can use WordPress, you can use VZ Story Teller.

= Can I export my scripts? =

Currently, you can print your scripts using the print-friendly template. PDF and Final Draft export are planned for future releases. Check our roadmap for upcoming features.

= Is this plugin free? =

Yes! VZ Story Teller is completely free and open source. There are no hidden fees, no subscriptions, and no premium versions.

= Can multiple people work on the same script? =

Yes! The plugin uses WordPress user management, so multiple users can collaborate on scripts. Each user needs appropriate permissions to edit scripts.

= What browsers are supported? =

VZ Story Teller works on all modern browsers including Chrome, Firefox, Safari, and Edge. It requires JavaScript to be enabled.

= Can I use this on my production site? =

VZ Story Teller is currently in version 1.0.0. While it's stable and functional, we recommend testing in a staging environment first and keeping regular backups of your scripts.

= Does it work with page builders? =

VZ Story Teller creates its own custom post type and editor interface, so it works independently of page builders. You can use both on the same site without conflicts.

= Can I customize the formatting? =

The plugin uses industry-standard screenplay formatting. Customization options are planned for future releases.

= Is it compatible with WooCommerce? =

Yes! VZ Story Teller works alongside WooCommerce and other WordPress plugins without conflicts.

= How do I get support? =

For support, please visit our GitHub repository or email support@melisaviroz.com. We also welcome bug reports and feature requests.
